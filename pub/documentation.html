Getting Started

DailyCheckIn library is a calendar generating library which is depended on jQuery.

The calendar is generated automatically based on the current month and the current date. It will change every day and month by itself.

One can either use the "check in" button or click on the date to check in.

When hovering over checked in days, previous records of that day will show up if exists.

Basic Setup

            
                var checkin = new Checkin()
                checkin.init()
                checkin.events()

API

Property Default Type Description
elements $(".checkin") HTMLdivElement A CSS selector for the HTMLdivElement itself (e.g. document.querySelector(".checkin")) that will be used for the calendar generation.
width 450px Int The width of the calendar.
height 450px Int The height of the calendar.
background '#d8efff' String The background color of the calendar.
radius 10 Int The border radius of the calendar.
color '#000000' String The text color of the calendar.
padding 10 Int The padding of the calendar.
dataArray / Array It stores the dates that the user checked in of this month.
records / Dictionary It stores the recorded texts from previous checked in days.
week Array ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'] The texts that represent days of a week.

Functions

Function Argument Behaviour Return Value
Checkin / This is the constructor of DailyCheckIn. /
init / It initiates a new calendar and adds to the HTML body. /
changeBackground color It changes the background color of the calendar to the specified color. /
changeCheckedColor color It changes the color of the checked dates to the specified color. /
changeTodayColor color It changes the color of the current date to the specified color. /
changeBtnColor color It changes the color of the "check in button" to the specified color. /
changeCalColor color It changes the color of the normal dates, which are neither checked in nor can be checked in, to the specified color. /
modal / It sets up the popup that occurs when checked in. /
events / It adds eventListeners to all the buttons. /
